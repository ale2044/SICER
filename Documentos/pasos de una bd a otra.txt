Vamos a utilizar las tablas EMPRESA, FAMIBA y TITUBA, ESTCONTA
1- Copio la base de datos del server.

2- Al abrir la bd el usuario en "juan" y la contraseña vacia. Luego pide contraseña que a la contraseña en "morseto".

3- En phpmyadmin crear una tabla que se llame "exportacion", preferiblemente que este vacía. en utf32_general_ci.

4- Previamente configurado Mysql Connector/ODBC (panel de control/herramientas administrativas/configurar orígenes de datos (ODBC)) cuando lo abrimos en "DNS usuario" tiene que tener los siguientes archivos TCP/IP 127.0.0.1 port 3306 user root Pass: admin. En "Database" seleccionamos "exportacion". Apretamos Test para saber si todo salio bien y listo.

5- Volvemos a la BD de access y en las tablas FAMIBA, EMPRESA y TIUTBA hacer lo siguiente: Clic derecho Exportar -> "Guardar como tipo" -> "ODBC" -> ".." -> Origen de datos de equipo -> "MySql" y aceptar.

6- Las tablas ya estarán pasadas a mysql, ir a phpMyAdmin y ahí en la tabla "exportacion" vamos a la pestaña "Exportar", exportamos y nos hará un archivo exportacion.sql.

7- Abrimos ese archivo y empezamos a modificar según corresponda cada tabla.
